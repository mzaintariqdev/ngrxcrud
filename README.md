# 📘 NgRx CRUD Demo (Angular)

## 📌 Summary

This is a simple **front-end CRUD demo project** built with **Angular and NgRx**.

It demonstrates basic Create, Read, Update, and Delete operations using a mock API and NgRx state management.

The project focuses on managing application state in the frontend using NgRx Store, Effects, Reducers, and Selectors.
